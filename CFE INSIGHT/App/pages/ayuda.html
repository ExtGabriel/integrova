Información para Solicitudes de Soporte

Para brindarte la mejor atención, hemos preparado una guía detallada sobre la información necesaria para procesar tu solicitud de manera eficiente.

Información Personal y Empresarial

  • Nombre completo del solicitante
  • Nombre de la empresa u organización
  • Cargo o puesto en la organización

Datos de Contacto

  • Correo electrónico de contacto
  • Número de teléfono (con lada)
  • Horario preferencial para contacto

Clasificación de Solicitud

  • Tipo de solicitud
  • Técnica (problemas con el sistema)
  • Informativa (dudas o consultas)
  • Comercial (cotizaciones o ventas)
  • Queja o reclamo

Descripción Detallada

  • Descripción completa del problema o solicitud
  • Pasos para reproducir el issue (si aplica)
  • Resultados esperados vs resultados actuales

Evidencia y Soporte

  • Capturas de pantalla del problema
  • Archivos adjuntos relevantes
  • Videos demostrativos (si aplica)

Información Técnica

  • Navegador web utilizado
  • Dispositivo y sistema operativo
  • Versión del navegador

Contexto Temporal

  • Fecha y hora del problema
  • Frecuencia del issue (primera vez, recurrente)
  • Tiempo estimado del impacto

Referencias del Sistema

  • Número de expediente relacionado
  • ID de formulario de auditoría
  • Módulo o sección específica del sistema
