Guía orientativa

Objetivo

Documentar el uso del muestreo estadístico o no estadístico (a juicio) para proporcionar una base razonable para las conclusiones sobre la población de la que se selecciona la muestra.

Las ISA relevantes relacionadas con este documento incluyen: ISA 330, ISA 500, ISA 530.

Este documento está diseñado para ayudar al auditor a generar tamaños de muestra para procedimientos de auditoría sustantiva. El modelo y sus cálculos asociados se proporcionan únicamente con fines orientativos, por lo que los usuarios deben aplicar un criterio profesional considerable.

Se recuerda a los usuarios que de acuerdo con la NIA 530 – Muestreo de auditoría, el auditor debe determinar los medios apropiados para seleccionar los elementos a probar a fin de reunir suficiente evidencia de auditoría apropiada para cumplir con los objetivos de los procedimientos de auditoría.

Los usuarios deben tener en cuenta que se requieren decisiones para diversos componentes del modelo. Estas incluyen, entre otras:

  • Decidir el nivel a fijar para los factores de riesgo asociados a un nivel de riesgo Alto/Medio/Bajo;
  • Decidir el nivel en que se establecerán los factores de riesgo asociados a "Otras pruebas sustantivas" (véase más adelante);
  • Decidir el monto de error tolerable;
  • Decidir sobre el monto de la incorrección esperada;
  • Determinación del tamaño de la población después de permitir exclusiones;
  • Elegir el nivel de confianza requerido para cada afirmación; y
  • Elegir un nivel apropiado de riesgos inherentes y de control.

Debe reconocer las limitaciones inherentes del modelo debido al redondeo y a los supuestos de "mejores prácticas". Estos supuestos incluyen limitar el porcentaje de errores esperados/tolerables y establecer el factor de riesgo para riesgos inherentes y de control elevados.
